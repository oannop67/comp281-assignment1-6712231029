## แนวคิด

- วาดทิวทัศน์ที่มี **ครบ 7 องค์ประกอบ** (ท้องฟ้า, ภูเขา, พระอาทิตย์, ท้องนา, ต้นไม้, บ้าน, แม่น้ำ)  
- ไม่ใช้ `drawImage()` และไม่ใช้ไลบรารี
